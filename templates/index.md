# Knowledgebase

An [Open Knowledge Format](https://github.com/GoogleCloudPlatform/knowledge-catalog/blob/main/okf/SPEC.md) v0.1 bundle. `kb-init.sh` regenerates this root index; it is reproduced here only as a reference for the expected shape.

`index.md` is a RESERVED file — a directory listing for progressive disclosure, with NO frontmatter. Links begin with `/` and are relative to the bundle root.

- [Sessions](/sessions/index.md) — dated session-handover journal.
- [Concepts](/concepts/index.md) — curated decisions, components, and open items.
- [Log](/log.md) — chronological change history.
